# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 0.x (latest) | Yes |
| older | No |

agenteval is pre-1.0; only the latest released version receives fixes.

## Reporting a vulnerability

Please report security issues **privately** — do not open a public GitHub issue.

**Email:** antonio@zaitoun.dev

Include a clear description, steps to reproduce, and impact assessment if known.

## Disclosure

- We aim to acknowledge reports within **5 business days**.
- We will work with you on a fix and coordinate public disclosure.
- Please allow up to **90 days** before public disclosure so a patch can ship.

We appreciate responsible disclosure and will credit reporters when appropriate.
